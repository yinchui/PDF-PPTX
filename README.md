# PDF转PPTX

将PDF格式的PPT转换为可编辑的PPTX文件的网页应用。

## 功能特点

- 纯前端处理，无需上传文件到服务器
- 支持拖拽上传PDF文件
- PDF页面预览
- 将PDF转换为可编辑的PPTX文件

## 使用方法

1. 打开 `index.html`
2. 拖拽或选择PDF文件
3. 预览PDF页面
4. 点击"转换为PPTX"按钮
5. 下载生成的PPTX文件

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- pdf.js (PDF解析)
- pptxgenjs (PPTX生成)
